# P5X Asset Tool

A simple tool for processing `.bundle` files from a selected directory and saving the modified files to an extraction directory, preserving the original folder structure.

## How to Use

1.  **Run the application.**
2.  Select the **Bundles Directory** containing the `.bundle` files you want to process.
3.  Select the **Extraction Directory** where you want to save the modified files.
4.  The tool will then process all `.bundle` files and save them to the extraction directory.

## Automated Builds

This project uses GitHub Actions to automatically build a Windows executable (`.exe`) for every commit pushed to the `main` branch.

You can find the latest builds on the [Releases page](https://github.com/SerenoV7/p5x-asset-tool/releases).
